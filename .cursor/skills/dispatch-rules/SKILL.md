# 派工與衝突規範

- **一人一廠原則**：同一人同日出現在多個廠區，強制標示為「紅色衝突」。
- **衝突處理**：禁止自動結算紅色衝突，必須由 PM 手動「認領」單一廠區，未解決前不可請款。
- **PY-SR 結構**：資料彙整與報表輸出必須以 PY (Project) 為主幹。
- **視覺系統**：
  - 綠燈：符合規則。
  - 黃燈：不滿 4h (待確認)。
  - 紅燈：一人多廠 (待人工裁決)。
